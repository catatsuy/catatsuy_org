pixivインターンに参加してきた

Posted on 2013/01/19

pixiv インターンに参加していました

軽く振り返りたいと思います

概要

今回のインターンは 2 週間という期間でした
私はどういう判断が下されたのかは知りませんが， pixiv の内部の API を追加することになりました

pixiv ではものすごく大雑把に言うと PHP で作ってある部分と PHP で JSON を返してそれを Ruby on Rails で表示している部分があります

今回はその PHP で JSON を返すという部分をやりました

すでに pixiv 内で作られた重厚なソースコードを読まないと開発を進められず，苦労しました

発表

最後にインターン中に何をやったのか発表しました

その時の写真を使って軽く説明します

今回の目標です（詳細は伏せます）

このあと具体的にやったことや苦労したところなどを説明しました

それで今回はソースコードを pixiv の Github のプライベートレポジトリに上げたので

こんな感じで終わりました

それで実際のコードはどんな感じよ？ と言われたので git の diff を表示して説明しているところです

評判は上々だったのでよかったです

オフィス

オフィスは代々木駅から少し歩いたところにあります

近いうちに移転するとかしないとか言っていました

入ってすぐのエントランスから pixiv っぽさがあってワクワクします

そしてオフィスに入ると広々とした開放的な会議室的なスペースがあります

先ほどの発表をやったのもここですし，会議はもちろん，勉強会や LT でもここが使われます

そしてこのスペースには

こういうのもあってとても pixiv らしくていい感じです

また先ほどの写真をよく見ると机に絵がかいてあることが分かると思いますが，こういう机はまだあって

こんな机で会議をしていることもあります

中に入っていくとオフィスに裸のサーバが置いてあったりします

最後にオフィス風景です

全員 pixiv が大好きで pixiv を作っていこうとしていることがすごく伝わって来ました
とても楽しかったです！ ありがとうございました！
